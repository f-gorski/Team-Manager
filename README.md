## Team-Manager concept application
Project of web application of a management panel of a sports team which allows users to join different sports sections within the team and communicate with other users and trainers via implemented messaging system and also to check their training schedules in a calendar module. Apllication provides 3 different levels of user permissions: user, trainer and admin. Admin permissions allow to assign Trainers, Trainers are allowed to manage the teams and set events in the calendar for the Users to check.

## Completed features
* Authentication with Passport.js
* Sports group module which allows to create teams and lets the users to join them.
* Messaging module which allows to send messages to other specified users.
* Calendar module that provides functionality of adding and displaying scheduled events in the calendar
* Permission levels

## Technologies
